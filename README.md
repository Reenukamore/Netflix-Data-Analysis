# Project Overview
This project focuses on analyzing Netflix’s content library using Excel, Oracle SQL, and Power BI. The goal of the project is to clean, analyze, and visualize Netflix data to uncover meaningful insights related to content trends, audience targeting, and platform growth.

#Tools & Technologies Used

Excel – Used for data cleaning, formatting, handling missing values, and preparing the dataset for analysis.
Oracle SQL – Used for storing the data, performing data validation, and answering analytical business questions through structured queries.
Power BI – Used to create interactive dashboards and visualizations for better interpretation of insights.
Dataset Source – Kaggle (Netflix Dataset)

#Data Cleaning & Preparation

Cleaned and formatted raw Netflix data in Excel.
Handled missing and inconsistent values.
Imported cleaned data into Oracle SQL.
Verified data integrity by checking record counts, null values, and duplicates.

#Analysis Performed

The following analyses were performed using Oracle SQL:
Distribution of Movies and TV Shows on Netflix
Year-wise growth of Netflix content
Monthly content release trends
Top contributing countries
Content rating distribution
Most popular genres on Netflix
Directors with the highest number of titles
Analysis of recent movie releases
Identification of the oldest content on Netflix
Country-specific analysis (India)

#Power BI Visualization

The same analysis was replicated in Power BI to build interactive dashboards, allowing:
Visual comparison of Movies vs TV Shows
Trend analysis using charts and graphs
Country-wise and genre-wise insights
Easy exploration of Netflix content patterns

#Conclusion

This project demonstrates an end-to-end data analysis workflow using Excel for data preparation, Oracle SQL for data analysis, and Power BI for visualization, helping uncover key insights into Netflix’s content strategy and growth trends.
